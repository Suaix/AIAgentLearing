## 5. 薄弱点清单

| 薄弱点 | 严重度 | 改进建议 |
|--------|--------|---------|
| Python 数据结构直觉（dict key 选型） | 🟡 轻微 | 写代码前先写下"key 是什么→value 是什么"的草稿 |
| `ThreadPoolExecutor.submit()` 传参语法 | 🟡 轻微 | 记住：`submit(fn, *args, **kwargs)` 而非 `submit(fn(...))` |
| `{val}` 被 Python 解析为 set | 🟡 轻微 | 有冒号 `{k:v}` 才是 dict，无冒号 `{v}` 是 set |

> 以上薄弱点均为 Python 语法层面，不涉及 AI Agent 核心概念，不影响后续模块学习。
