## 遇到的问题与解决

1. **.env 路径错误导致 401**：`load_dotenv()` 写错了路径参数 → 无参数调用修复
2. **中文引号导致 Python 语法错误**：`print("...")` 中中文双引号 `"` 冲突 → 改用单引号
3. **极难题目标注错误**：字母 'o' 数量标注为 19，实际为 12 → 模型答对，标注错误，已修正
4. **CoT 实验"无差异"**：模型能力过强导致 CoT 效果不明显 → 正确归因，理解 CoT 的条件性

## 关键收获

- Few-shot 的效果 = 示例质量 × 示例多样性 − 示例偏差
- In-Context Learning 不是训练——模型权重不变，退出对话后效果消失
- CoT 不是让模型变聪明，是让推理过程变成可见 token 约束答案生成
- CoT 的价值 = f(任务难度, 模型能力)——差距越大，CoT 越有用
- 强模型（DeepSeek V4 Flash）对大部分中等难度题不需要显式 CoT
