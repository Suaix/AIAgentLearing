## 遇到的问题与解决

1. **tool_calls 消息顺序错误**：DeepSeek API 要求每个 tool role 消息之前必须有对应的 assistant tool_calls 消息。解决：将 assistant 消息（包含所有 tool_calls）放在 for 循环之前一次性追加。
2. **`call_with_timeout` 的 `TimeoutError` 捕获**：确认 Python 3.12 中 `concurrent.futures.TimeoutError` 是 `builtins.TimeoutError` 的别名，`except TimeoutError` 可用。
